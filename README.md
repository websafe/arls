arls
====


Examples
--------


### Recursive directory listing ordered by size sorted ascending


~~~~ bash
arls /home/user size
~~~~

is exactly the same as:

~~~~ bash
arls /home/user s
~~~~

or

~~~~ bash
arls /home/user s
~~~~

or

~~~~ bash
arls /home/user bytes
~~~~

or

~~~~ bash
arls /home/user b
~~~~



### Recursive directory listing ordered by modification time sorted ascending


~~~~ bash
arls /home/user size
~~~~

is exactly the same as:

~~~~ bash
arls /home/user s
~~~~

or

~~~~ bash
arls /home/user s
~~~~

or

~~~~ bash
arls /home/user bytes
~~~~

or

~~~~ bash
arls /home/user b
~~~~
